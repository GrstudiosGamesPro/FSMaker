Introducción

Esta es la documentación de FSMaker, una herramienta para programar con componentes y eventos en Unity.

Objetivo

FSMaker tiene como objetivo proporcionar a los desarrolladores de Unity una forma visual y fácil de programar la lógica de sus juegos utilizando componentes y eventos.

Características principales

  • Interfaz intuitiva para la programación visual.
  • Facilita la creación de comportamientos complejos mediante la conexión de componentes y eventos.
  • Elimina la necesidad de escribir código manualmente para muchas tareas comunes en Unity.
  • Compatible con las versiones más recientes de Unity y sus características nativas.

Requisitos previos

Para utilizar FSMaker en tu proyecto de Unity, asegúrate de tener instalado Unity en tu sistema.
